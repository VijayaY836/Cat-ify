# Cat-ify

A Manifest V3 Chrome extension that replaces every image on a webpage
with a random cat photo from the cataas.com API.

## Install (Developer Mode)
1. Clone this repo
2. Go to chrome://extensions
3. Enable "Developer mode" (top right)
4. Click "Load unpacked" and select this folder
5. Visit any webpage — images will be replaced with cats automatically
6. Click the toolbar icon → "Re-shuffle Cats" to fetch new ones without reloading
